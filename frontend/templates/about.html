About SafeURL

SafeURL uses a machine learning model trained on real-world data to detect whether a given URL is safe or potentially malicious.

🚨 Features:

  • Detect phishing links using ML
  • Custom URL feature extraction
  • Trained on real data
  • Instant results

🛠️ Built With:

  • Python, Flask
  • HTML, CSS
  • Scikit-learn, Pandas
